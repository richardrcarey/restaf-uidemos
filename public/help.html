CAS Upload Parameters

If the action is tables.upload then the parameters should as shown below. You should replace the values for casli, name and filetype as appropriate. You can add the other options of the upload actions as needed.

{ "qs": { "action": "upload" }, "data": { "output": { "casout": { "caslib": "casuser", "name": "upload1" }, "importOptions": { "fileType": "csv" } }, "input": {} } }
